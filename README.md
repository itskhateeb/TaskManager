# TaskFlow - Team Task Manager

A full-stack team task management application with role-based access control (Admin/Member), project management, and task tracking.

## 🚀 Features

- **Authentication System** - Secure signup/login with JWT
- **Role-Based Access** - Admin and Member roles with different permissions
- **Project Management** - Create, update, delete projects and manage team members
- **Task Management** - Create, assign, track tasks with status and priority
- **Dashboard** - View tasks, statistics, overdue tasks, and personal assignments
- **Admin Panel** - User management system for admins

## 🛠️ Tech Stack

### Backend
- Node.js & Express.js
- MongoDB with Mongoose
- JWT for authentication
- bcryptjs for password hashing

### Frontend
- React 18 with Vite
- Tailwind CSS for styling
- Axios for API calls
- React Router DOM for navigation

## 📋 Prerequisites

- Node.js (v14 or higher)
- MongoDB Atlas account (or local MongoDB)
- npm or yarn package manager

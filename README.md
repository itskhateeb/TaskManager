# TaskFlow - Team Task Manager

A full-stack team task management application with role-based access control (Admin/Member), project management, and task tracking.

## ✨ Features

### For All Users
- **Secure Authentication** - JWT-based login and signup
- **Dashboard** - View task statistics and personal assignments
- **Task Status Updates** - Change task status (Pending, In Progress, Completed)

### For Admin Users
- **Project Management** - Create, edit, delete projects
- **Task Management** - Create, assign, edit, delete tasks
- **User Management** - Add/remove users via Admin Panel
- **Full Access** - View all projects and tasks

### For Member Users
- **View Assigned Tasks** - See only tasks assigned to them
- **Update Status** - Change status of their own tasks
- **View Projects** - See projects they are part of
